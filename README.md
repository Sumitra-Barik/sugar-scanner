Smart Sugar Scanner

Demo web dashboard for a school/college tech function.

Files:
- index.html — main UI
- styles.css — styles and animations
- app.js — demo logic (no backend)

How to run:
- Open `index.html` in your browser (double-click in Explorer) or run a simple local server:

  For Python 3:

  ```bash
  python -m http.server 8000
  ```

  Then open: http://localhost:8000/index.html

Notes:
- Scanning shows randomized demo products from a small set.
- All data is stored in memory for the session only.
